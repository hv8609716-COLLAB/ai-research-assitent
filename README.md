# 📄 AI Research Assistant (Flask + Python)

An AI-powered Research Assistant built using **Python** and **Flask** that allows users to upload a PDF document and ask questions or generate a summary from its contents.

---

## 🚀 Features

- 📂 Upload PDF files
- 📖 Extract text from PDF
- 🤖 Ask questions based on the uploaded document
- 📝 Generate a document summary
- 🌐 Simple Flask Web Interface
- ⚡ Fast and lightweight

---

## 🛠️ Technologies Used

- Python 3
- Flask
- PyPDF2 / pypdf
- HTML
- CSS (Optional)

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/AI-Research-Assistant.git
```

Go inside the project

```bash
cd AI-Research-Assistant
```

Install dependencies

```bash
pip install flask pypdf
```

---

## ▶️ Run the Project

```bash
python 123.py
```

Open your browser

```
http://127.0.0.1:5000
```

---

## 📌 How to Use

1. Start the Flask server.
2. Open the browser.
3. Upload a PDF document.
4. Enter your question.
5. Click **Ask**.
6. The assistant will answer based on the uploaded PDF.
